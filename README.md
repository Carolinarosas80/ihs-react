# Inter High Supply — React + Vite

## Instalación
```bash
npm install
npm run dev
```

## Imágenes — colocar en /public
```
public/
├── logodoradoihs.jpg     ← logo dorado (navbar + footer)
├── avionCarga.jpg        ← avión de carga o depósito (hero + global)
├── reunioncorpo.jpg      ← foto sección About
├── mundodorado.jpg       ← mapa dorado sección Global
├── duty1.jpg             ← Duty Free (markets)
├── crucero.jpg           ← Cruceros (markets)
├── frontera.jpg          ← Fronterizas (markets)
└── feria.jpg             ← Ferias (markets)
```

## Deploy Vercel
1. Subir a GitHub
2. Importar en vercel.com
3. Framework: Vite (autodetectado)
